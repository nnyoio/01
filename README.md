걍 좋아하는 노래 공유하는 사이트입니다 :D

# 주요 기능

- 노래 검색: iTunes Search API로 곡 제목 · 아티스트 검색
- 앨범 커버: 검색 결과에 앨범 아트 표시
- 30초 미리듣기: 사이트 내에서 바로 재생
- Spotify 바로가기: Spotify에서 전체 곡 듣기
- 사용자 시스템: 아이디 · 비밀번호로 계정 생성, 각자 피드 관리
- 구경 모드: 로그인 없이도 전체 피드 열람 가능
- 함께 듣는 중: 2명 이상 추가 시 모아서 보기

# 사용된 것들

프론트엔드 : HTML · CSS · Vanilla JS
스타일 : Tailwind CSS (CDN)
폰트 : 온글잎 은별 (눈누에서 다운로드 받기)
음악 검색 : iTunes Search API (spotify 대체 가능)
